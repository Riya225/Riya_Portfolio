# Frontend Portfolio

This repository contains a static portfolio website built with HTML and CSS.

## Files included

- `index.html` — portfolio page markup
- `styles.css` — responsive styling and layout
- `profile.svg` — profile placeholder image
- `README.md` — instructions for using and customizing the portfolio

## How to view locally

Open `index.html` directly in your browser.

If you prefer a local server, you can use one of these commands:

- Python 3: `python -m http.server 8000`
- Node.js: `npx serve .`

Then open `http://localhost:8000`.

## Customize the portfolio

- Replace `profile.svg` with your own photo or illustration.
- Edit the hero text and introduction in `index.html`.
- Add or update your real projects in the Projects section.
- Adjust the skills list and education entries to reflect your experience.
- Tweak colors, spacing, and typography in `styles.css`.

## Notes

This is a static site with no build step, so it can be hosted anywhere that serves HTML files.
